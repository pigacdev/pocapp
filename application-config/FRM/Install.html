3. Postavke classloadera:
Potrebno je podesiti classloader za aplikaciju (NE OSTAVITI DEFAULT postavke):

EAR:

  • Classloader mode: PARENT_LAST
  • WAR Classloader policy: APPLICATION

WAR:
  • Classloader mode: PARENT_LAST

5. Shared libraries:
Potrebno je dodijeliti shared library aplikaciji "???-ear". Molim da se smisleno napravi direktoriji na filesystemu npr: "D:\resources\???"
Iz isporuke kopirati "resources/okoline/../app.properties" konfiguracijsku datoteku u kreirani direktoriji filesystem te ažurirati konfiguracijsku datoteku ovisno o oklini na koju se aplikacija deploya.

6. JVM argumenti:
Alikacija traži formatiranje iznosa za hrvatski jezik (što je ispravno i sa strane poslovne logike) koji bi defaultno trebao biti setiran na operativnom sustavu ili u JVM argumentima WebShere-a: u segment WebSphere Admin Console > Servers > Server types > WebSphere  application servers > [IME_SERVERA] > Java and Process Management >  Process definition > Java Virtual Machine > Generic JVM arguments   potrebno je dodati "-Duser.language=hr -Dusercountry=HR"
